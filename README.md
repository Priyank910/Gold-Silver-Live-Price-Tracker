# 🏆 Gold & Silver Live Price Tracker

A Django-powered web application designed to monitor and display gold and silver prices through an intuitive dashboard interface.

The project provides dedicated views for precious metal tracking and demonstrates full-stack web development using Python and Django.

---

## 🚀 Features

### 📈 Precious Metal Tracking

* Gold Price Monitoring
* Silver Price Monitoring
* Dedicated Commodity Dashboards
* Structured Financial Data Display

### 👤 Authentication Pages

* Login Interface

* User Account Pages

### 📊 Dashboard Experience

* Organized Financial Dashboard

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Priyank910/Gold-Silver-Live-Price-Tracker.git
```

### Navigate Into Project

```bash
cd Gold-Silver-Live-Price-Tracker
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install django
```

---

## ▶️ Run Application

Apply migrations:

```bash
python manage.py migrate
```

Start development server:

```bash
python manage.py runserver
```

Visit:

```text
http://127.0.0.1:8000/
```
